# Luna — Perimenopause Tracker

A free, privacy-first period and symptom tracker designed specifically for perimenopause.

## About
Luna was built to fill a gap in the market — most period tracking apps are designed for 
regular cycles. Luna is built from the ground up for the irregular, unpredictable experience 
of perimenopause.

## Features
- Daily logging — flow, symptoms, triggers, mood, energy, sleep and hydration
- Cycle predictions that account for irregular and missed periods
- Hormone pattern detection and PMDD awareness
- HRT and medication tracking
- Printable GP report
- Custom tags
- Health data CSV import (Oura, Garmin, Apple Health export)
- Evidence-based Learn articles
- 100% private — all data stays on your device,
